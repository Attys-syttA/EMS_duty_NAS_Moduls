This repo contains the EMS Duty Discord bot and supporting scripts. A modular refactor exists under `EMS_Duty_Moduls`.

To run the modular core, make sure .env contains the following:
```
BOT_FILE=EMS_Duty_Moduls/core.py
```

See `EMS_Duty_Moduls/README.md` for instructions on creating and hot-reloading command modules.
